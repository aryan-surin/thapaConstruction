# Video Assets

Place your optimized video files here:

- `construction-hero.webm` (Primary, <2MB)
- `construction-hero.mp4` (Fallback, <4MB)

See `/VIDEO_HERO_SETUP.md` for detailed setup instructions.

## Quick Start

1. Download construction-themed video from [Pexels](https://www.pexels.com/videos/)
2. Optimize using FFmpeg or online tools
3. Place files in this directory
4. Test hero section on homepage

**Note**: Videos are NOT tracked in git due to file size. Add them manually after cloning the repository.
